# Pyramids Mart — Starter Dashboard (Zaad-style brief adapted)

This is a starter scaffold for **Pyramids Mart** (dashboard + backend).
Colors and logo are tailored to the uploaded logo.

Folders:

- `frontend/` — React + Vite + Tailwind + Framer Motion starter

Follow steps in each folder's README to run locally.
